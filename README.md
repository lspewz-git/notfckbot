# NotFckBot — Телеграм-бот для отслеживания серий

Этот бот позволяет подписываться на ваши любимые сериалы и получать уведомления о выходе новых серий. Данные берутся через API Кинопоиска.

## Основные функции
- 🔍 **Поиск сериалов**: Найдите любой сериал по названию.
- 📺 **Мои подписки**: Управляйте списком своих подписок.
- ⚡️ **Уведомления**: Бот проверяет обновления каждые 3 часа и присылает сообщение, если вышла новая серия.
- 👥 **Поддержка групп**: Работает как в личных чатах, так и в группах.

## Технологии
- Node.js + Telegraf (Telegram Bot API)
- Sequelize ORM + MariaDB
- Docker + Docker Compose
- Axios (API Integration)
- node-cron (Scheduling)

## Как запустить

### 1. Подготовка
Создайте файл `.env` на основе `.env.example` и заполните его:
```env
BOT_TOKEN=ваш_токен_от_BotFather
KINOPOISK_API_KEY=ваш_ключ_от_poiskkino.dev
```

### 2. Запуск через Docker (Рекомендуется)
Убедитесь, что у вас установлен Docker и Docker Compose. Выполните команду:
```bash
docker-compose up -d --build
```
Это запустит базу данных MariaDB и само приложение. База данных будет сохраняться в volume `mariadb_data`.

### 3. Запуск локально (для разработки)
1. Установите зависимости:
   ```bash
   npm install
   ```
2. Убедитесь, что у вас запущена MariaDB и настройки в `.env` корректны.
3. Запустите приложение:
   ```bash
   node src/index.js
   ```

## Команды
- `/start` — Показать главное меню и инициализировать чат.
- `/help` — Показать справку.
- Используйте Reply-кнопки для навигации.

### Тестирование уведомлений
Для ручной проверки уведомлений используйте:
*   `test_all.bat` — Единый скрипт для выбора типа теста (Обычная серия, Весь сезон, Первая серия сезона).
2. В контейнере или локально выполните:
   ```bash
   # Внутри Docker:
   docker-compose exec app node src/test-notifications.js <KP_ID>
   
   # Локально:
   node src/test-notifications.js <KP_ID>
   ```
   Скрипт временно «откатит» состояние сериала в базе и запустит проверку обновлений.

## Структура проекта
- `src/models/` — Модели базы данных (Chat, Series, Subscription).
- `src/api/` — Клиент для работы с API Кинопоиска.
- `src/bot.js` — Логика бота и обработка команд.
- `src/cron.js` — Фоновая задача для проверки обновлений.
- `src/db.js` — Инициализация базы данных.
